## 📋 如何使用这些动画

### 方法 1：在现有角色节点上配置

如果场景中已有角色节点（包含 PlayerController 或 EnemyAI 组件）：

1. **选中角色节点**
2. **在 Animation 组件中**：
   - 点击 **Clips** 右边的 **+**
   - 从 `assets/animations/player/` 拖入所有动画
   - 设置 **Default Clip** 为 `player_idle`
3. **调整缩放**：
   - 设置节点或 `facingVisualRoot` 的 **Scale**
   - 玩家推荐：**0.1**
   - 史莱姆推荐：**0.08**

---

### 方法 2：创建新的角色节点

如果需要创建新的角色：

1. **创建节点**
   - 在场景中创建一个新节点
   - 添加 **Sprite** 组件
   - 添加 **Animation** 组件
   - 添加 **PlayerController** 或 **EnemyAI** 组件

2. **配置 Sprite**
   - SpriteFrame：选择任意一个精灵帧（如 `spriteFrame_0`）
   - 或留空，让 Animation 控制

3. **配置 Animation**
   - 按方法 1 的步骤添加动画

4. **调整缩放**
   - Scale: 0.1 (玩家) 或 0.08 (史莱姆)

---
